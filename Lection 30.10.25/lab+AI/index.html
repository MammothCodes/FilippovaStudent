Контрольные вопросы

1. Какие значения могут принимать элементы массива?

Элементы массива в JavaScript могут принимать любые значения:

  • Примитивные типы: числа, строки, булевы значения, null, undefined, symbol, bigint
  • Объекты: объекты, другие массивы, функции, даты, регулярные выражения
  • Смешанные типы: массив может содержать элементы разных типов одновременно
  • Динамическое изменение: тип элементов может меняться во время выполнения программы

Примеры:

let mixedArray = [1, "текст", true, null, {name: "объект"}, [1,2,3], function() {}];
