Контрольные вопросы

1. В чем разница между var и let?

var - функция-скопленная переменная, поднимается (hoisting), может быть переопределена

let - блок-скопленная переменная, не поднимается, не может быть переопределена в том же блоке

2. Какие типы данных в JavaScript вы знаете?

Примитивные: string, number, boolean, undefined, null, symbol, bigint

Объектные: object, array, function

3. В каком типе сохраняется переменная, полученная с помощью команды prompt?

Переменная сохраняется как string (строка), даже если пользователь ввел число.
